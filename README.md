# AI-Tools 🤖

AI-Tools 是一个专为 AI Agent 开发和使用设计的全能管理工具。它旨在简化多个 AI 引擎（如 Claude Code, Gemini CLI, Antigravity, OpenCode 等）之间的 Skill (技能) 同步与资源管理。

## 🌟 核心功能

- **🚀 一键同步 (Link)**: 自动将你的 Skill 源码分发挂载到所有主流 AI Agent 工具的配置目录中。
- **🔄 强制更新 (Update)**: 深度扫描所有 AI 相关目录，自动识别 Git 仓库或 Skill 源码，并强制同步远端（自动处理冲突）。
- **🔍 状态监控 (List)**: 清晰展示当前各工具加载的 Skill 列表、指向路径以及链接有效性。
- **🧹 智能清理 (Cleanup/Remove)**: 轻松移除不再需要的技能，或一键清理失效的“断头”软链接。
- **💬 交互模式**: 支持持续交互式命令行，操作更直观。

## 📂 支持的工具

AI-Tools 已深度适配以下 AI Agent 工具：
- **Claude Code** (`~/.claude`)
- **Gemini CLI** (`~/.gemini`)
- **Antigravity** (`~/.antigravity`)
- **OpenCode** (`~/.opencode`)
- **主流 AI IDE/工具** (Cursor, Trae, Windsurf, CherryStudio 等的插件更新支持)

## 🛠 安装与使用

### 1. 克隆仓库
```bash
git clone git@github.com:ly87ing/ai-tools.git
cd ai-tools
chmod +x ai-tools.sh
```

### 2. 交互模式 (推荐)
直接运行脚本即可进入交互式控制台：
```bash
./ai-tools.sh
```

### 3. 命令行快捷操作
- **更新所有 AI 资源**: `./ai-tools.sh update`
- **挂载新的 Skill**: `./ai-tools.sh link ~/path/to/your/skills`
- **查看已挂载列表**: `./ai-tools.sh list`
- **移除特定技能**: `./ai-tools.sh remove <skill_name>`

## 💡 开发初衷

随着 AI Agent 工具的增多，每个工具都有自己的 Skill 定义规范和存储目录。手动同步这些技能既繁琐又容易出错。AI-Tools 建立了一套基于 `SKILL.md` 识别的通用管理机制，让开发者能够“一次编写，到处运行”。
